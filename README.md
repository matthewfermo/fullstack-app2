This responsive fullstack application allows users to enter the name of the player they'd like to track along with their stats such as:

- Minutes Played
- Goals
- Assists

## Technologies Used

- HTML
- JavaScript
- CSS
- EJS Templating
- Node.js
- MongoDB

## Installation

1. Clone repo
2. run `npm install`

## Usage

1. run `node server.js`
2. Navigate to `localhost:8080`
